# Airways
It allows the user to do airline ticket booking
It asks the user to choose between first class or economy class.
Then it aks the user to enter the number of tickets.
If the number exceeds the number of available seats, then he is asked whether he wants to check 
in the other class.
If yes, then he is directed to that class and continued with the booking.
The program then asks the user details like name, flight and time.
With all these details, a boarding pass is generated.
